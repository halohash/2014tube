# 2014Tube

A recreation of the classic 2014 YouTube experience.

This project aims to replicate the look and feel of YouTube as it was in 2014 — including layout, styling, and core UI behavior.

---

## 🌐 Live Demo

https://halohash.github.io/2014tube/

---

## 📦 Features

* 2014 YouTube-style layout
* TV page (`/tv`)
* Classic styling and UI elements
* Static frontend served via Node.js and Github Pages
* Simple Express server for hosting

---

## 🚀 Running Locally

### 1. Clone the repo

```bash
git clone https://github.com/halohash/2014tube.git
cd 2014tube
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the server

```bash
npm start
```

### 4. Open in browser

http://localhost:2014

---

## 🛠️ Deployment (Render)

This project is configured to run as a Web Service on Render.

**Settings:**

* Build Command: `npm install`
* Start Command: `node server.js`
